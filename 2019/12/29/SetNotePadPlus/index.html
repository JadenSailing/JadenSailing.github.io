如何设置Notepad++执行Python(Lua)

发表于 2019-12-29 分类于 other

Notepad++的运行可以方便的使用外部程序运行当前文件。对于Python和Lua这种脚本语言，可以直接运行查看结果。
如我们要添加Python脚本的快捷运行：


指令如下，这里采用的python3.8

cmd /k D:\python-3.8.0\python-3.8.0-embed-amd64\python.exe "$(FULL_CURRENT_PATH)" & PAUSE & EXIT

cmd /k 表示使用命令行执行程序，执行完不关闭
python3.8路径
“$(FULL_CURRENT_PATH)” 表示当前文件
还可以保存快捷键如图

之后Alt+Shift+L即可以python执行当前文件
