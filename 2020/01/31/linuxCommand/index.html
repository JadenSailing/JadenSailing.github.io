1 ls和ll
均为列出当前目录命令，区别

ls横排，ll竖排。

2 cd
切换文件夹，可以多层跳入

cd dir
cd d1/d2/d3
cd .. 表示跳回父路径

3 mkdir
创建文件夹
4 rm
移除文件或文件夹

rm -r表示删除目录

5 zip
压缩文件指令，一般最常用的是递归的压缩当前文件夹所有内容到某个压缩文件下

zip -q -r data.zip data 我一般使用这个 意思是压缩当前路径下的data文件夹
-q表示不显示详细信息，-r表示递归压缩子目录

6 sz rz
用于linux和windows上下载文件，本地的上下载目录可以在SecureCRT里设置，如图

一般的，查询Linux服务器上的日志可以简单的利用上述指令下载到本地查看。
不过相对来说直接在Linux上查询过滤会更方便，而且速度更快。

8 head tail

可以配合cat使用

cat file |head -n 20 |tail +n 10 显示10-20行的内容
